Get Started with Claude Code on the Web

A beginner-friendly guide to using AI to write and modify code - no prior coding experience needed.

What is Claude Code on the Web?

Claude Code on the web is a tool that lets you use AI to work with code stored on GitHub. Think of it as having an expert coding assistant that can:

  • Read and understand code - Ask questions about how code works or how features are built
  • Fix bugs - Describe a problem and Claude will find and fix it
  • Make changes - Request new features or modifications to existing code
  • Run tests - Claude can test its changes to make sure they work
  • Work while you're away - Start a task and come back when it's done

No coding knowledge required - you describe what you want in plain English, and Claude handles the technical work.

Before You Begin

This guide will walk you through setting up Claude Code on the web step by step. Here's what you'll need:

  • A Claude account with Pro, Max, or Team/Enterprise access (free accounts don't include this feature)
  • A GitHub account (free to create - we'll cover this)
  • A code repository on GitHub that you want to work with (or create a new one)
