Data Tables for Sensitivity Analysis

Learn to test how profit changes across a range of uncertain values

1 The Sensitivity Analysis Challenge

Santa Rosa Raisins must buy grapes at an uncertain market price (between $0.15 and $0.35/lb). How much does profit change across this range? Let's use a Data Table to find out!

The Question

What if market price is $0.15? $0.20? $0.30? $0.35?
Data Tables calculate ALL scenarios at once—no manual testing required!

6 Configure Column Input Cell

In the Data Table dialog, enter the Column input cell (because test values are in a column). This is the cell that should be replaced with each test value—the Market Grape Price cell (B2).

Column vs. Row Input

Column input: Test values are in a vertical column (most common)
Row input: Test values are in a horizontal row
For one-variable analysis, only fill in one!

Business Insight

Even with market prices at the worst case ($0.35/lb), SRR remains profitable. The $32,500 profit swing across the range is manageable risk. This sensitivity analysis gives confidence in the pricing strategy!
