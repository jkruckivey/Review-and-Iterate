IEC Expected Value Calculations

Calculate probability-weighted revenue for uncertain broadcast territories

Expected Value Formula

For each client, calculate expected value accounting for both primary renewal AND backup (if primary doesn't renew):

EV = (Renewal_Prob × Renewal_Fee) + ((1 - Renewal_Prob) × Backup_Prob × Backup_Fee)

Key insight: Backup revenue is conditional—it only happens if the primary doesn't renew, so we multiply by (1 - Renewal_Prob).
