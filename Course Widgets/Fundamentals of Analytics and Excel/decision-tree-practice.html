Step 4: Calculate Expected Values

Using backward induction, let's calculate the expected value for each strategy.

Strategy A Expected Value

EV(A) = (0.40 × $200,000) + (0.30 × $80,000) + (0.30 × -$20,000)
EV(A) = $80,000 + $24,000 - $6,000
EV(A) = $98,000

Strategy B Expected Value

EV(B) = (0.60 × $100,000) + (0.30 × $50,000) + (0.10 × $10,000)
EV(B) = $60,000 + $15,000 + $1,000
EV(B) = $76,000

Net Expected Values (after $50,000 investment)

Net EV(A) = $98,000 - $50,000 = $48,000
Net EV(B) = $76,000 - $50,000 = $26,000
Net EV(No Investment) = $0 - $0 = $0
Understanding the calculations: We multiply each outcome by its probability, then sum those weighted values to get the expected value. This represents the average outcome if we could repeat this decision many times.
