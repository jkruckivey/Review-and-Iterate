Design System Style Comparison

Live-rendered components across three visual approaches. Compare and decide which system to standardize on — or which elements to mix.

1. Buttons

Primary action, secondary action, and scenario toggle (preset) — the three button types every widget uses.

Option A — Geist Neutral Current standard
Near-black fill, 8px radius, Geist 600. Presets use pastel teal active state. No gradients.
Option B — Rounded / Soft
Deep navy fill, pill-shaped (24px radius), Inter font. Presets use indigo active state.
Option C — High Contrast / Sharp
Black fill, 4px radius, uppercase labels, heavy border on secondary. Maximum legibility.
