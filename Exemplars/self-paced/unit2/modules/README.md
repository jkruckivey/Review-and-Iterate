# Unit 2 Module Storyboards

**Course:** Introduction to Strategic Analysis and Action (Self-Paced)
**Unit:** 2 - Goal Setting
**Structure:** 4 modules (Intro → Content → Practice → Wrap-Up)
**Total Time:** 4.5–5.5 hours

---

## Module Structure

Unit 2 contains 4 modules organized in this directory:

1. **module-1-intro-goal-setting.md** — "What Mountain Do You Want to Climb?" (50–65 min)
   - Learning outcomes: Understand industry structure's impact on performance
   - Key activities: Industry performance visualization, learning outcomes checklist
   - Assessment: Knowledge check on industry effects concept

2. **module-2-five-forces.md** — "The Five Forces That Shape Industry Attractiveness" (60–75 min)
   - Learning outcomes: Apply Five Forces framework to analyze industries
   - Key activities: Five forces videos, diagnostic frameworks, knowledge checks
   - Assessment: 5-question scenario-based quiz

3. **module-3-practice-application.md** — "Apply Five Forces to Real Industries" (70–90 min)
   - Learning outcomes: Demonstrate competency through independent analysis
   - Key activities: Two interactive widgets, Canary Medical interview, guided reflection
   - Assessment: Reflection with AI coach feedback + optionally instructor review

4. **module-4-wrap-up.md** — "Synthesize Learning and Bridge Forward" (30–40 min)
   - Learning outcomes: Consolidate knowledge; prepare for Unit 3
   - Key activities: Key takeaways summary, recommended readings, bridge to strategic positioning
   - Assessment: Reflection on how goal-setting informs positioning choices

---

## Module Learning Outcomes (ALOs)

| Module | ALO Focus | Specific Outcomes |
|--------|-----------|-------------------|
| **1: Introduction** | Understand | Explain why industry structure affects firm performance; recognize that industry effects explain ~20% of variation |
| **2: Five Forces** | Know & Analyze | Identify five competitive forces; analyze any industry using structured framework; explain how each force affects profitability |
| **3: Practice & Application** | Apply & Demonstrate | Use Five Forces to analyze specific industries; set realistic goals based on industry constraints; articulate strategic implications |
| **4: Wrap-Up** | Synthesize | Summarize key concepts; connect goal-setting to value proposition and positioning; preview next unit |

## Element Type Legend

Used throughout module storyboards:

- **▶ Video** — Instructor video or interview excerpt
- **▬ Text** — Prose content (readable in 2-3 minutes)
- **🖼 Image** — Diagram, infographic, photo, chart
- **⚙ Widget** — Interactive tool or calculator
- **◈ AI Chat** — AI coach or chatbot interaction
- **📝 Discussion/Assessment** — Quiz, reflection, discussion prompt
- **▼ Accordion** — Optional expandable deep-dive content

## Key Accessibility Features

All modules include:

✅ **Video captions** — VTT files for all 4 instructor videos + interview
✅ **Alt text** — Descriptive labels for all diagrams and charts
✅ **Keyboard navigation** — All widgets fully operable with Tab + Enter
✅ **Screen readers** — ARIA labels on interactive elements
✅ **High contrast** — 4.5:1 minimum on text
✅ **Focus indicators** — Visible 3px outlines on focusable elements
✅ **Responsive design** — Works on 6-inch to 27-inch screens

**WCAG Standard:** 2.2 AA
**ATAG Standard:** 2.0 AA (platform side)
**Tested with:** NVDA (Windows), VoiceOver (Mac), keyboard-only navigation

## Cohort Comparison

These self-paced modules are adapted from the cohort storyboard Module 1 (Introduction to Goal Setting). Key changes:

| Aspect | Cohort | Self-Paced |
|--------|--------|-----------|
| **Modules** | 8 (includes bridge, welcome, executive, etc.) | 4 (intro, content, practice, wrap-up) |
| **Time** | 7–9 hours | 4.5–5.5 hours |
| **Videos** | 1 executive interview + instructor videos | 2 instructor videos + 1 interview |
| **Widgets** | 4–5 per module | 1–2 per module |
| **Readings** | Optional, sprinkled throughout | Consolidated in wrap-up |
| **Assessment** | 3–4 summative checks + milestone | 4 formative checks only |

**Philosophy:** Self-paced modules are streamlined for autonomy and clarity, not reduced rigor.
